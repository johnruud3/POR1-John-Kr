CSS Frameworks Course

Visit live site View GitHub README

This project was part of my CSS frameworks course, where the goal was to build a fully responsive website using modern CSS techniques and framework-style patterns. I focused on creating a clean layout that works well on mobile, tablet, and desktop screens.

Throughout the project I practiced structuring pages with semantic HTML, organizing styles into reusable components, and adding interactive elements such as navigation, cards, and form layouts. I also learned how to plan a design system with consistent spacing, typography, and color usage.

One of the biggest takeaways was understanding how responsive design improves usability. By testing breakpoints and adjusting layouts, I gained confidence in building websites that feel polished and professional across different devices.
