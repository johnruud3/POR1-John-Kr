Project Exam 1

Visit live site View GitHub README

Project Exam 1 was a major milestone in my learning journey. It was the first time I combined HTML, CSS, and JavaScript into a complete, working project. I had to plan the layout, write clean markup, and add interactivity through JavaScript all at the same time.

Working on this exam taught me how important it is to structure your code before you start writing it. I learned about DOM manipulation, event listeners, and how to connect logic to the user interface in a meaningful way.

Looking back, this project gave me the confidence to tackle more complex assignments. It showed me that even with limited experience, I could build something real and functional and that pushed me to keep going and learning more.
