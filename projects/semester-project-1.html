Semester Project 1

Visit live site View GitHub README

Semester Project 1 was my first large, structured assignment where I had to plan and deliver a complete multi-page website on my own. The brief was to build a website for a community science museum, targeting a younger audience with an engaging and accessible design.

This project introduced me to professional workflows: using GitHub for version control, creating a project board to track tasks, and breaking the work into manageable milestones. It was my first real experience with planning before coding and it made a huge difference to the quality of the final result.

I built the site using only HTML and CSS, focusing on accessibility, semantic markup, and responsive layouts. It taught me how much you can achieve without JavaScript when you understand the fundamentals well.
