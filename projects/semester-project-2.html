Semester Project 2

Visit live site View GitHub README

Semester Project 2 was my most advanced project so far an auction website where users can browse listings, place bids, and manage their profiles. The project combined front-end development with API integration and user authentication.

I built features such as listing creation, search and filtering, bid placement, and profile management. Working with live data from an API taught me how to handle loading states, error messages, and form validation in a real-world context.

Collaboration and planning were also key parts of this project. I used GitHub for version control, followed a structured workflow, and tested thoroughly before deployment. The result is a functional auction platform that demonstrates my ability to deliver a complete web application from concept to deployment.
