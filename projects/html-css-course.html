HTML and CSS Course

Visit live site View GitHub README

The HTML and CSS course was where it all began. This was my very first web development project, and the goal was to build a fictional clothing store called Rainy Days using only HTML and CSS no JavaScript at all.

I spent a lot of time on this project understanding how the box model works, how to position elements, and how to create layouts that look good on different screen sizes. Every detail from the navigation bar to the product cards was my first time ever getting to know how to code as structured as possible with just the basics of HTML and CSS. That was a good feeling.

Looking back, this project represents the foundation of everything I know today. It taught me to be patient, methodical, and to enjoy the process of building something from nothing. The skills I developed here have been the backbone of every project I've worked on since.
